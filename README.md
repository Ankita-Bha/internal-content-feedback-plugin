# Internal Content Feedback Plugin

A WordPress plugin that adds a private internal feedback meta box to posts and pages for editorial workflows.

## Features
- Adds a custom admin meta box to posts and pages
- Stores feedback securely using WordPress post meta
- Nonce-based validation and capability checks
- Feedback is never visible on the public site

## Tech Stack
- PHP
- WordPress Hooks (`add_meta_boxes`, `save_post`)
- WordPress Post Meta API

## Use Case
Useful for teams that need to leave internal editorial notes without exposing them publicly.

## Development
Built and tested using WordPress Playground.
